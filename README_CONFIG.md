# コンテンツ設定ガイド

## 概要

`config.js`ファイルで、音声、字幕、アバター画像、背景画像を一元管理できます。

## 設定ファイルの構造

### デフォルト設定 (`DEFAULT_CONFIG`)

全スライドで共通の設定を定義します。

```javascript
const DEFAULT_CONFIG = {
    basePath: 'data/',              // リソースファイルのベースパス
    defaultAvatars: {               // デフォルトのアバター画像
        man: {
            static: 'man.png',       // 静止画
            animated: 'man.gif'      // アニメーション（GIF）
        },
        woman: {
            static: 'wman.png',
            animated: 'wman.gif'
        }
    },
    defaultBackgrounds: [           // デフォルトの背景画像リスト
        'background_01.png',
        'background_02.png',
        'background_03.png'
    ],
    defaultSubtitleDuration: 2000   // デフォルトの字幕表示時間（ミリ秒）
};
```

### スライド設定 (`SLIDE_CONFIG`)

各スライドごとの設定を定義します。

```javascript
const SLIDE_CONFIG = {
    1: {
        // 背景画像（省略可能：省略時はデフォルトから循環使用）
        background: 'background_01.png',
        
        // アバター画像（省略可能：省略時はデフォルトを使用）
        avatars: {
            man: {
                static: 'man.png',
                animated: 'man.gif'
            },
            woman: {
                static: 'wman.png',
                animated: 'wman.gif'
            }
        },
        
        // 音声ファイル（必須）
        audio: {
            man: 'man-01.wav',
            woman: 'woman-01.wav'
        },
        
        // 字幕データ（必須）
        subtitles: {
            man: [
                {text: "字幕テキスト1", duration: 2000},  // durationはミリ秒
                {text: "字幕テキスト2", duration: 3000}
            ],
            woman: [
                {text: "字幕テキスト1", duration: 2000}
            ]
        }
    }
};
```

## 新しいスライドの追加方法

1. `SLIDE_CONFIG`オブジェクトに新しいスライド番号を追加
2. 必要なリソースファイルを`data/`フォルダに配置
3. 設定を記述

例：
```javascript
4: {
    background: 'background_01.png',
    audio: {
        man: 'man-04.wav',
        woman: 'woman-04.wav'
    },
    subtitles: {
        man: [
            {text: "新しいセリフ1", duration: 2000},
            {text: "新しいセリフ2", duration: 3000}
        ],
        woman: [
            {text: "新しいセリフ1", duration: 2500}
        ]
    }
}
```

## リソースファイルの配置

```
data/
├── man.png              # 男性アバター静止画
├── man.gif              # 男性アバターアニメーション
├── wman.png             # 女性アバター静止画
├── wman.gif             # 女性アバターアニメーション
├── background_01.png    # 背景画像1
├── background_02.png    # 背景画像2
├── background_03.png    # 背景画像3
├── man-01.wav           # 男性音声（スライド1）
├── woman-01.wav          # 女性音声（スライド1）
├── man-02.wav           # 男性音声（スライド2）
├── woman-02.wav          # 女性音声（スライド2）
└── ...
```

## 省略可能な設定

- **背景画像**: 省略時は`defaultBackgrounds`からスライド番号に応じて循環使用
- **アバター画像**: 省略時は`defaultAvatars`を使用
- **字幕のduration**: 省略時は`defaultSubtitleDuration`を使用（文字列形式の場合）

## 字幕データの形式

### オブジェクト形式（推奨）
```javascript
{text: "字幕テキスト", duration: 2000}
```

### 文字列形式（簡易）
```javascript
"字幕テキスト"  // durationはデフォルト値を使用
```

## 字幕表示の仕組み

このシステムでは、**1行表示＋マーカー方式**で字幕を表示します。

### 表示方法
- 各キャラクター（男性/女性）の全字幕テキストを1行に結合して表示
- 音声の再生時間に応じて、現在話している部分を自動的にハイライト（マーカー表示）
- マーカーは青色の背景色で表示され、視覚的に現在のセリフを識別できます

### 動作の流れ
1. 音声再生開始時に、そのキャラクターの全字幕を1行で表示
2. 音声の再生時間（`currentTime`）に基づいて、該当する字幕セグメントを自動判定
3. 該当する部分をマーカーでハイライト表示
4. 音声再生終了時に字幕表示をクリア

### 字幕のタイミング調整
各字幕セグメントの`duration`は、音声の実際の再生時間と同期するように設定してください。マーカーは`startTime`と`endTime`を累積計算して自動的に判定されます。

## システム機能

### 自動再生モード
右上の自動再生ボタン（緑色の再生アイコン）をクリックすると、全スライドを自動的に連続再生します。

- 各スライドの音声再生が終了すると、自動的に次のスライドに進みます
- 最後のスライド終了後は、最初のスライドに戻ってループ再生します
- 停止ボタン（赤色の停止アイコン）で自動再生を停止できます

### テーマ切り替え機能
右上のテーマ切り替えボタンで、ダークモードとライトモードを切り替えられます。

- デフォルトはダークモードです
- ボタンをクリックするたびに、ライトモードとダークモードが切り替わります
- 設定はセッション中に保持されます

### 音声プリロード機能
システムは自動的に次のスライドの音声を事前に読み込みます。

- 現在のスライド表示時に、次のスライドの音声をバックグラウンドでプリロード
- これにより、スライド切り替え時の音声再生がスムーズになります
- プリロード済みの音声はキャッシュされ、再利用されます

### 再生制御機能

#### 再生/一時停止
- **再生ボタン**（右上の青いボタン）: 現在のスライドの音声を再生/一時停止
- **スペースキー**: キーボードショートカットで再生/一時停止を切り替え
- 一時停止中は、キャラクターのアニメーションも静止画に切り替わります
- 再開時は、一時停止した位置から続きを再生します

#### スライドナビゲーション
- **前へ/次へボタン**: スライドを手動で切り替え
- スライド切り替え時は、現在の再生を停止して新しいスライドを表示します

### 字幕マーカーの同期
字幕マーカーは音声の再生時間と完全に同期します。

- 一時停止時: マーカー更新を停止（現在の位置を保持）
- 再開時: 保持された位置から自動的に同期を再開
- 音声の`currentTime`に基づいて、リアルタイムでマーカー位置を更新

## 注意事項

- 音声ファイル名は`basePath`からの相対パスで指定
- 画像ファイル名も`basePath`からの相対パスで指定
- スライド番号は連番である必要はありませんが、`totalSlides`の値は自動的に`SLIDE_CONFIG`のキー数から計算されます
- 字幕の`duration`は、実際の音声の再生時間と一致させることを推奨します（マーカー表示の精度が向上します）
- 音声ファイルが見つからない場合、システムは自動的にデモ用のビープ音にフォールバックします
